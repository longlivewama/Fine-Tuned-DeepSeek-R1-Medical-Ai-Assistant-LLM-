# Fine-Tuned-DeepSeek-R1-Medical-Ai-Assistant-LLM-
FEMOZ AI is a specialized Large Language Model (LLM) fine-tuned to act as an intelligent medical consultant. Leveraging the DeepSeek-R1-Distill-Llama-8B architecture and optimized via Unsloth, this system utilizes Chain-of-Thought (CoT) reasoning to process complex clinical vignettes.
